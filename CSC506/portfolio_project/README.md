# Modular Portfolio Project: Algorithms, Sets, and Data Structures

## Project Structure

```text
portfolio_project_modular/
│
├── main.py
├── README.md
└── portfolio_system/
    ├── algorithms/
    │   ├── sorting.py
    │   └── selection.py
    ├── data_structures/
    │   ├── custom_set.py
    │   ├── stack.py
    │   ├── queue_ds.py
    │   ├── hash_table.py
    │   ├── binary_search_tree.py
    │   └── graph.py
    ├── analysis/
    │   └── performance.py
    └── ui/
        └── menu.py
```

## Why this version is better
- Each concern is separated into its own module.
- The code is easier to maintain, test, and explain.
- It looks more like a real software project instead of a single script.
- The UI, algorithms, data structures, and performance analysis are clearly separated.

## How to Run
From the project root, run:

```bash
python main.py
```
