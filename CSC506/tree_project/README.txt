Project: Binary Search Tree and BST-Based Map

Files:
- bst_node.py: defines the BST node
- binary_search_tree.py: implements BST operations
- bst_map.py: implements a map using the BST
- list_map.py: simple list-based map for performance comparison
- test_driver.py: runs demonstrations and tests

How to run:
1. Place all files in the same folder.
2. Open a terminal in that folder.
3. Run: python test_driver.py

This program demonstrates:
- BST insertion, searching, deletion
- in-order, pre-order, and post-order traversals
- min and max retrieval
- tree balance detection
- BST-based key-value storage
- performance comparison with a list-based map
- visual tree structure before and after deletions
