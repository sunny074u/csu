Graph System Project Files

Files included
1. adjacency_matrix_graph.py
   - Graph implementation using an adjacency matrix
2. adjacency_list_graph.py
   - Graph implementation using an adjacency list
3. test_graph_system.py
   - Test driver that builds the sample graph, runs DFS, BFS, shortest path, and manipulation steps
4. test_graph_output.txt
   - Sample console output from the test program
5. graph_system_project_report.docx
   - Submission-ready report with performance comparison, analysis, visual representation, and references

How to run
1. Place all three Python files in the same folder.
2. Open a terminal in that folder.
3. Run:
   python test_graph_system.py

What the program demonstrates
- Graph construction for both representations
- Adding and removing vertices and edges
- Depth-first search with step-by-step output
- Breadth-first search with step-by-step output
- Dijkstra shortest path with step-by-step output
- Graph display in matrix/list form and ASCII visual form
